# 📌 Overview

**BuildStack AI** is a premium AI-assisted website builder that allows users to:

- Generate websites instantly from prompts  
- Preview generated websites in real-time  
- Save projects to database  
- Edit generated HTML live  
- Manage projects through dashboard  
- Delete / Update projects  
- Secure routes using JWT authentication  
- Experience modern dark premium UI  

This project demonstrates **Full Stack Engineering + Product Thinking + Scalable Architecture**.

# 🚀 Installation

## Clone Repository

```bash
git clone <your-repo-link>
cd BuildStack-AI
```

## Frontend Setup

```bash
cd client
npm install
npm run dev
```

## Backend Setup

```bash
cd server
npm install
npm run dev
```

## Configure Environment Variables

Create `.env` inside `server`:

```env
DATABASE_URL="your_mysql_connection_string"
JWT_SECRET="your_secret_key"
PORT=5000
```
